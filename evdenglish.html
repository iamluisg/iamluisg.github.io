ROLE: iOS Developer

ABOUT THE PROJECT

While working abroad as an ESL instructor, I saw the need for resources covering specific topics and could not find what I was looking for, so I decided to create my own and put it into an easy to use app.

I developed the UI for the app, designed the icons, structured the data in an xml file, and completely coded the app from scratch in Objective-C. The app is designed to be a reference for students, so the data is available offline and is organized in a manner that allows certain information to be found quickly and easily.
