Client Project

What Color Top

ABOUT THE PROJECT

What Color Top is a medical reference app designed to help lab technicians quickly find the correct lab collection tubes for various tests. The goal was to create a streamlined, easy-to-use tool that delivers accurate information at the point of need. I led the app’s development from initial concept to App Store release—structuring the data model, implementing the core logic, and building the interface based on provided design mockups. I also handled testing and refinement to ensure a smooth user experience across devices.
